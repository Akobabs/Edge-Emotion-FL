In this demonstration layer, the trained global emotion classification model runs **entirely client-side** in your browser using **ONNX Runtime Web**. No image pixels leave your local computer. 🛡️ Research Highlights: Handling Heterogeneity & Defending Privacy

The charts above demonstrate the key research achievements of the simulation. By applying **FedProx** on client nodes, the global model converges smoothly despite highly Non-IID emotional data. Crucially, our server-side **LBAAFedAvg** filter successfully flags and discards anomalous weights submitted by compromised clients in every round, protecting the global model from stealth poisoning attacks without degrading benign accuracy!

⚠️ CORS Policy Warning / Local Server Required

Browsers block file loading via the `file://` protocol due to security policies. To see real-time webcam inference and load simulation charts, please start a quick local server in this directory:
python -m http.server 8000
Then open http://localhost:8000 in your web browser.
